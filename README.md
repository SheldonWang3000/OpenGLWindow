| [![Build Status](https://travis-ci.org/euangelion666/OpenGLWindow.svg?branch=master)](https://travis-ci.org/euangelion666/OpenGLWindow) | Linux / macOS (gcc / clang) |

Using [GLM 0.9.9](https://github.com/g-truc/glm)
